# 03 镜像仓库

镜像仓库负责存储、分发和管理容器镜像，是容器化交付和 Kubernetes 发布流程中的关键基础设施。

本章围绕常用镜像仓库、阿里云镜像仓库、Harbor 私有仓库安装与使用展开，重点掌握镜像推送、拉取、私有仓库认证、`insecure-registry` 配置、权限管理、镜像清理和镜像复制。
