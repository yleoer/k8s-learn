# 镜像制作

第 02 章已完成容器核心概念、Docker 架构和基础操作的记录。镜像是容器运行的模板，因此镜像制作是从“使用容器”走向“交付应用”的关键环节。

本章从 Dockerfile 基础指令开始，逐步覆盖构建缓存、分层优化、多阶段构建和多架构镜像发布，并通过附录补充业务镜像制作范例。这些记录为后续镜像仓库、Kubernetes 应用部署和 CI/CD 发布流程提供可交付的镜像基础。

本章涵盖以下内容：

- Dockerfile 基础指令与编写原则
- 容器启动命令与构建参数
- 文件复制、工作目录与运行用户
- 镜像分层、缓存与体积优化
- 多架构镜像构建与发布
- 业务镜像制作附录
