## 默认约定

| 项目 | 默认值 |
| --- | --- |
| 操作系统 | Ubuntu 24.04 LTS，代号 `noble` |
| Kubernetes | v1.36.x |
| 容器运行时 | containerd |
| CNI 网络插件 | Calico |
| 安装方式 | kubeadm |
| Pod 网段 | `10.244.0.0/16` |
| Service 网段 | `10.96.0.0/12` |

如果修改 Kubernetes 版本、Pod 网段或镜像源，后续所有相关配置要同步调整。尤其是 kubeadm 的 `--pod-network-cidr` 必须和 Calico IPPool 的 `cidr` 一致。
