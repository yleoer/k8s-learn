# 入门起步

本章是全书的起点，目标是从零搭建一套可运行、可验证、便于反复练习的 Kubernetes 实验集群。当前阶段以基础环境、运行时组件和集群初始化为前置背景。

本章围绕 kubeadm、containerd、Calico 和基础排障链路展开，形成从主机准备到集群验证的完整路径。完成本章后，后续 Pod、Service、Ingress、存储、调度和可观测性等章节才具备稳定实验环境。

本章涵盖以下内容：

- 节点角色、版本选型与网络规划
- Ubuntu 节点基础环境配置
- containerd 与 Kubernetes 组件
- kubeadm 集群初始化与节点加入
- Calico 网络插件与集群网络
- 集群验证与常见故障排查
- Docker 并行使用与执行速查
