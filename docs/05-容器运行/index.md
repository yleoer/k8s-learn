# 容器运行

第 04 章完成了 Harbor 私有仓库的部署和镜像推拉链路，镜像进入 Kubernetes 节点后，还需要由节点上的容器运行时完成拉取、解包和容器启动。

本章从集群视角下沉到节点内部，围绕 CRI、containerd、客户端工具、运行时命名空间和私有仓库访问配置展开；这些内容为后续理解 kubelet、Pod 生命周期和节点排障提供运行时基础。

本章涵盖以下内容：

- CRI 与 containerd
- 客户端工具与命名空间
- 镜像仓库配置
- 镜像与容器管理
- 排障链路与面试要点
