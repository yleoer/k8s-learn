# 镜像仓库

第 03 章完成了容器镜像的构建与多架构制作，镜像需要进入统一的存储与分发位置，才能被 Kubernetes、CI/CD 流水线和不同运行环境稳定拉取。

本章围绕镜像仓库的基本概念展开，逐步说明 Harbor 私有仓库的部署、镜像推拉、权限控制和运维管理；这些内容为后续集群运行时、应用发布和持续交付流程提供镜像基础。

本章涵盖以下内容：

- 镜像仓库基础
- Harbor 安装部署
- Harbor 镜像管理
- Harbor 权限管理
- Harbor 运维管理
