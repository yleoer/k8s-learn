# 容器基础

第 01 章已经用 kubeadm + containerd 把 K8s 集群跑起来了，Pod 内部本质上是容器。本阶段从 Docker 命令行入手，先把容器运行、镜像管理、日志查看、文件拷贝和服务部署这些基础能力练熟，为后续理解 K8s 的 Pod、容器运行时、镜像拉取和应用部署打基础。本阶段的任务是：

- 区分物理机、虚拟机和容器的运行方式与使用场景。
- 理解镜像、容器、镜像仓库、镜像地址和版本标签的基本概念。
- 明确 Docker 组件架构，以及 Docker 与 Kubernetes 容器运行时的关系。
- 掌握 Docker 基础命令、系统信息查看和镜像管理操作。
- 熟悉容器创建、启动、停止、删除、日志查看和进入容器等日常操作。
- 使用数据卷和文件拷贝完成容器数据持久化与文件交换。
- 按通用步骤部署一个容器化服务，形成可复用的操作流程。
