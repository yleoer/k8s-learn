# 容器基础

第 01 章已通过 kubeadm 与 containerd 完成了 Kubernetes 集群的搭建。Pod 内部的本质是容器，因此理解容器的运行原理与操作方式，是深入掌握 Kubernetes 的必要前提。

本章从 Docker 命令行入手，依次覆盖容器运行、镜像管理、日志查看、文件拷贝和服务部署等核心操作。完成本章后，后续 Pod 调度、容器运行时、镜像拉取和应用部署将具备直观基础。

本章涵盖以下内容：

- 容器运行模型与核心概念
- Docker 组件架构与运行时关系
- Docker 基础命令与镜像管理
- 容器操作、日志与排障
- 容器数据持久化与服务部署
