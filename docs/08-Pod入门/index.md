# Pod 入门

第 07 章记录了 kubectl、Namespace 与基础资源操作。本章从 Pod 入手深入核心工作负载单元，进一步记录业务容器在集群中的运行方式。

本章围绕 Pod 的资源定义、容器配置、资源分配、生命周期、健康检查和优雅退出展开，形成一条从创建到排障、再到发布稳定性的完整实践路径。这些内容将为后续的 Deployment、Service、ConfigMap、Secret 和存储等章节提供基础支撑。

本章涵盖以下内容：

- Pod 资源定义与基础配置
- Pod 资源分配与调度排查
- Pod 环境变量与内置元数据
- Pod 镜像拉取与重启策略
- Pod 生命周期与优雅退出
- Pod 健康检查与探针配置
